## Description
 A simple druidic spell that causes roots to erupt from the ground.

## Quick Info
| Property | Value                        |
| -------- | ---------------------------- |

## Mechanics & Effects
- **Effect Details:**
    - **Attack**: Roll 1D2, add INT modifier for magic damage.
    - **Root**: Roots a single target for 1 turn, preventing movement.
    - **Resource Cost**: 2 MP
- **Range:**
```
[o][x][x]
```
